Hero dish photos go in THIS folder: /public/dishes/

The landing hero flip-cards look for a photo matching each dish's slug.
Accepted file types: .jpg (preferred), .png, .jpeg, or .webp.
If a photo is missing, the card's back face gracefully shows the dish
colour + name instead of a broken image.

Expected filenames (one per card):
  ethiopian-pour-over.jpg     -> Blue Tokai  / Ethiopian Pour Over
  ricotta-pancakes.jpg        -> Fig & Maple / Ricotta Pancakes
  cold-brew-float.jpg         -> Kopi Klub   / Cold Brew Float
  pistachio-tart.jpg          -> La Folie    / Pistachio Tart
  burrata-toast.jpg           -> Perch       / Burrata Toast
  miso-ramen.jpg              -> Naaru       / Miso Ramen
  salted-caramel-crepe.jpg    -> Suzette     / Salted Caramel Crêpe
  pulled-pork-slider.jpg      -> Smoke House / Pulled Pork Slider

Tip: roughly square / portrait crops look best (cards are ~180x145px,
photos are cropped with object-fit: cover).
